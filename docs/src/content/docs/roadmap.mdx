# Roadmap

Lightfast MCP is under active development. This page outlines our plans for the future.

## Current Focus

- Improving Blender integration stability
- Adding support for additional creative applications
- Enhancing documentation and examples

## Upcoming Features

- Support for TouchDesigner
- Support for Adobe Creative Suite
- Improved error handling and logging
- More comprehensive examples

## Long-term Vision

Our goal is to make AI-assisted creation accessible to everyone who uses creative software. We aim to build a robust, flexible system that can connect any AI model to any creative application through a standardized protocol.

For the latest roadmap updates, please visit our [GitHub projects page](https://github.com/lightfastai/lightfast-mcp/projects).
