# Examples

Learn how to use Lightfast MCP with these practical examples.

## Basic Blender Scene Information

Get information about the current Blender scene:

```python
import asyncio
from lightfast_mcp.servers.blender_mcp_server import get_state

async def get_blender_info():
    scene_info = await get_state()
    print(scene_info)
    
    # Parse the JSON string to get structured data
    import json
    data = json.loads(scene_info)
    
    # Print object names
    print("Objects in scene:", data.get("objects", []))
    
asyncio.run(get_blender_info())
```

## Executing Blender Python Code

Execute Python code directly in Blender:

```python
import asyncio
from lightfast_mcp.servers.blender_mcp_server import execute_command

async def create_cube():
    # Create a simple cube in Blender
    code = """
    import bpy
    
    # Delete default cube if it exists
    if 'Cube' in bpy.data.objects:
        bpy.data.objects.remove(bpy.data.objects['Cube'])
    
    # Create a new cube
    bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
    cube = bpy.context.active_object
    cube.name = 'AI_Generated_Cube'
    
    # Add a material
    mat = bpy.data.materials.new('AI_Material')
    mat.diffuse_color = (1, 0, 0, 1)  # Red
    cube.data.materials.append(mat)
    """
    
    result = await execute_command(code_to_execute=code)
    print(result)
    
asyncio.run(create_cube())
```

## Claude Client Integration

Example of using Lightfast MCP with Claude:

```python
# This example would be in your Claude-connected application
from mcp.client import Client

async def connect_to_blender():
    # Initialize MCP client
    client = Client()
    
    # Connect to Lightfast MCP server
    await client.connect("localhost:9877")
    
    # Get Blender state
    state = await client.get_state()
    
    # Execute a command in Blender
    result = await client.execute_command(
        code_to_execute="""
        import bpy
        
        # Create a simple scene
        bpy.ops.mesh.primitive_monkey_add(size=2, location=(0, 0, 0))
        monkey = bpy.context.active_object
        monkey.name = 'AI_Monkey'
        
        return "Created monkey successfully!"
        """
    )
    
    return result

# In your Claude integration, this function would be called
# when the AI decides to interact with Blender
```
